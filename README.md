# Stock-LLM Research Dashboard Demo

Public static demo for the Stock-LLM research dashboard.

- This is a point-in-time research dashboard, not an investment recommendation.
- High score is not buy_now.
- Quote helper 只是在瀏覽器上加一層報價顯示；公開 GitHub Pages 若沒有連到使用者本機工具，會以「報價暫無」提示，不能把空值當成即時價格。
- This repo is automatically refreshed from the private Stock-LLM_Wiki repository after dashboard checks pass.
